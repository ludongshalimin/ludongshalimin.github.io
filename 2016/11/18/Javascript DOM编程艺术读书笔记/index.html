Javascript DOM编程艺术读书笔记

2016-11-18

基础知识

什么是DOM：

　　document object module，文档编程模型
　　一个与系统平台和编程语言无关的接口，
　　程序和脚本可以通过这个接口动态的访问和修改文档的内容，结构和样式
在javascript中，所有的变量实际上都是某种类型的对象

对象的分类：

　　１内建对象：比如数组
　　２宿主对象：比如BOM和DOM
　　３用户自定义对象

DOM文档的结点类型

　　１元素结点
　　２文本结点
　　３属性结点

什么是平稳退化

　　让网站的访问者在不支持javascript的浏览器中，
　　或者在被禁用javascript的浏览器中能够正常的访问网页
　　一个经典的平稳退化的例子

什么是渐进增强

　　用一些额外的信息层去包裹原始数据
　　网页设计人员流传着这样一句话，内容就是一切，这话不完全对
　　应该说是：标记良好的内容就是一切
　　在给内容加上各种标记后，就可以使用各种CSS指令控制内容的显示效果　

从DOM角度如何提高性能

　　1尽量少的访问DOM和减少标记
　　2合并和放置脚本：多个脚本放置在一个文件，
　　　　　　　　　　把所有的放到文件的末尾，之前
　　3压缩脚本：不必要的字节，如空格和注释，统统删掉　　

Ajax

　　对页面的请求以异步的方式发送到服务器。
　　而服务器不会用整个页面来响应请求，它会在后台处理请求，
　　与此同时用户还能继续浏览页面并与页面交互
　　Ajax技术的核心是XMLHttpRequest对象，这个对象充当浏览器中的脚本（客户端）与服务器之间中间人的角色

Hijax

　　渐进增强的使用Ajax
　　拦截登陆表单的请求，让 XMLHttpRequest请求代为发送
　　如果用户禁用了javascript，登陆表单照样让用户登陆，只不过花的时间要长一些
