Javascript DOM编程艺术读书笔记

2016-11-18

基础知识

什么是DOM：

　　document object module，文档编程模型
　　一个与系统平台和编程语言无关的接口，
　　程序和脚本可以通过这个接口动态的访问和修改文档的内容，结构和样式
在javascript中，所有的变量实际上都是某种类型的对象

对象的分类：

　　１内建对象：比如数组
　　２宿主对象：比如BOM和DOM
　　３用户自定义对象

DOM文档的结点类型

　　１元素结点
　　２文本结点
　　３属性结点

什么是平稳退化

　　让网站的访问者在不支持javascript的浏览器中，
　　或者在被禁用javascript的浏览器中能够正常的访问网页
　　一个经典的平稳退化的例子

什么是渐进增强

　　用一些额外的信息层去包裹原始数据
　　网页设计人员流传着这样一句话，内容就是一切，这话不完全对
　　应该说是：标记良好的内容就是一切
　　在给内容加上各种标记后，就可以使用各种CSS指令控制内容的显示效果　

从DOM角度如何提高性能

　　1尽量少的访问DOM和减少标记
　　2合并和放置脚本：多个脚本放置在一个文件，
　　　　　　　　　　把所有的放到文件的末尾，之前
　　3压缩脚本：不必要的字节，如空格和注释，统统删掉　　

Ajax

　　对页面的请求以异步的方式发送到服务器。
　　而服务器不会用整个页面来响应请求，它会在后台处理请求，
　　与此同时用户还能继续浏览页面并与页面交互
　　Ajax技术的核心是XMLHttpRequest对象，这个对象充当浏览器中的脚本（客户端）与服务器之间中间人的角色

Hijax

　　渐进增强的使用Ajax
　　拦截登陆表单的请求，让 XMLHttpRequest请求代为发送
　　如果用户禁用了javascript，登陆表单照样让用户登陆，只不过花的时间要长一些

数据类型和引用类型之间的差别

2016-11-18

五种基本数据类型：Undefined ，Null， Boolean， Number ，String

五种引用类型： Object类型，Array类型，Data类型，RegExp类型，Function类型

什么是基本类型的值，什么引用类型的值

基本类型的值就是由基本类型所生成的变量
引用类型的值是指由引用类型所生成的变量
例如

基本类型：指的是保存在栈内存中的简单数据段
引用类型：指的是那些保存在堆内存中的对象，
　　　　　换句话说，变量名实际上是一个指针，而这个指针指向的位置就是对象的位置

两种不同的访问方式

　基本类型：按值访问，操作的是它们的实际值
　引用类型：按引用访问，当查询时，需要先从栈中读取内存地址，
　　　　　　然后按照指针所指的地方，找到堆内存里面的值

使用特点

　　１动态的属性
　　　　基本类型的值是不可以变得
引用类型的值是可以变的，可以为引用类型添加属性和方法，也可以删除属性和方法
　　２复制变量的值
　　　　如果从一个变量向另一个复制基本类型的值，会在变量对象上创建一个新值，
　　　　然后将该值复制到尾新变量分配的位置上
　　　　如果从一个变量向另一个变量复制引用类型的值，同样也会将存储在变量对象中的值复
　　　　制一份放到为新变量分配的内存空间中。不同的是，这个值实际上是一个指针，而这个
　　　　指针指向存储在堆中的一个对象。复制结束后，两个变量实际上将引用同一个对象。因
　　　　此改变其中一个变量，就会影响到另一个变量
　　３传递参数
　　　　ECMAScript中所有函数的参数，都是按值传递的。也就是，把函数外部的值复制给函数
　　　　内部的参数，就是把值从一个变量复制到另一个变量一样。基本类型的值传递如同基本
　　　　类型变量的复制一样。引用类型值的传递，则如同引用类型变量的复制一样。
例如

分析：上面例子可以看出来，如果person是按照引用传递的，那么person就会自动被修

　　　　　　　改为指向其name属性值为”haha”。但是，当接下来访问person.name时，显示的仍
　　　　　　　然是“sasa”。这说明即使在函数内部修改了参数的值，但原始的引用仍然保持不
　　　　　　　变。实际上，当函数内部重写obj时，这个变量的引用的就是一个局部对象了。而
　　　　　　　这个局部对象会在函数执行完后立即销毁。

判断类型的最佳的使用方法

1typedef操作符是判断基本类型的最佳工具
2如果变量是NULL或者对象，typedef将返回“object”
3instanceof用于检测引用类型，可以检测到具体的，它是什么类型的实例
4如果变量是给定类型的实例，instanceof操作符返回true
